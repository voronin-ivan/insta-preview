# Instant-Preview

This SMM-tool helps to preview instagram-profile with custom parameters (description, count of followers and etc.) in a few clicks.

## [Demo](https://instant-preview.com)

### Created with
- TypeScript
- React
- Redux

#### Install

```sh
$ npm install
```

#### Development

```sh
$ npm start
```

#### Lint

```sh
$ npm run lint
```

#### Build

```sh
$ npm run build
```
